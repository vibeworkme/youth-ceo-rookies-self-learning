# 청년사장 루키즈 ON 셀프러닝

청년사장 루키즈 ON 강의안을 초보자 자기주도 학습용 정적 웹사이트로 재구성한 워크북입니다.

## 주요 기능

- 브랜드 캐릭터, 문제 발견, AI 도구 활용, 아이디어 확장, 아이템 선정, 1분 발표문 순서 학습
- 초보자용 7단계 로드맵과 단계별 "따라 하기" 안내
- 단계별 완료 체크로 진행률 확인
- 예시 자동 채우기로 완성 흐름 미리 보기
- 단계별 입력값 브라우저 자동 저장
- 진행률 표시
- AI 프롬프트 복사
- 문제 렌즈와 해결 패턴 버튼
- 1분 발표문 자동 생성, 복사, 텍스트 저장

## 실행

정적 파일만으로 동작합니다.

```bash
python3 -m http.server 4173
```

브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:4173/youth-ceo-rookies-self-learning/
```
